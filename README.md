# Link Sphere Frontend

pnpm, Vite, React, TypeScript, Tailwind CSS, shadcn-ui 기반 프로젝트입니다.

## 시작하기

### 설치

```bash
pnpm install
```

### 개발 서버 실행

```bash
pnpm dev
```

### 빌드

```bash
pnpm build
```

### 미리보기

```bash
pnpm preview
```

### 린트

```bash
pnpm lint
```

### 포맷팅

```bash
pnpm format
```

## 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **shadcn-ui** - UI 컴포넌트
- **React Query** - 서버 상태 관리
- **React Router** - 라우팅
- **Zustand** - 클라이언트 상태 관리
- **React Hook Form** - 폼 관리
- **Zod** - 스키마 검증
